Endpoint Server Implemented with NodeJS + Express

This server implements endpoints for Users, Posts, and Comments.

Available Routes

[GET: /] response: index.html

[POST: /test] request: json-object, response: json-object

[GET: /users] response: { status, data }

[GET: /users/:id] request: { username, email, about }, response: { status, data }

[POST: /users] request: { username, email, about }, response: { status, data }

[PUT: /users/:id] response: { status, data }

[DELETE: /users/:id] response: { status, data }

[GET: /posts] response: { status, data }

[GET: /posts/:id] request: { title, author, content }, response: { status, data }

[POST: /posts] request: { title, author, content }, response: { status, data }

[PUT: /posts/:id] response: { status, data }

[DELETE: /posts/:id] response: { status, data }

[GET: /comment] response: { status, data }

[GET: /comment/:id] request: { user, content, post }, response: { status, data }

[POST: /comment] request: { user, content, post }, response: { status, data }

[PUT: /comment/:id] response: { status, data }

[DELETE: /comment/:id] response: { status, data }
